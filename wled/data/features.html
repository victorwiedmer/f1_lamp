Live Event Features

Automatic LED effects from live F1 telemetry. Requires a live timing connection.
🏆 Winner Celebration
Rainbow spinner on chequered flag. Lasts until the next state change.
💜 Fastest Lap Flash
3 s purple strobe when a driver sets the overall fastest lap.
✅ DRS Open Flash
Brief white pulse when DRS detection zones open. Disabled by default.
🔴 Start Lights
5 red LED segments light up one-by-one then go dark on session start.
😴 Deep Sleep
Outside race weekends the lamp sleeps (5 µA). Requires battery; disabled by default.
